# 🖨️ PrintFy: Skip the Line, Not the Print

PrintFy is the ultimate campus printing solution designed to save students from the agony of long queues and unreliable printing kiosks. By digitizing the workflow, we turn the tedious process of **"upload-wait-print"** into **"upload-choose-breeze."**

## ✨ Why PrintFy?

| Feature | The Problem | The PrintFy Solution |
| --- | --- | --- |
| Lightning Fast | Wasted time in physical lines | Upload documents in seconds, print instantly. |
| Skip the Queue | Not knowing when a machine is free | Select your arrival time and secure your slot. |
| Secure Payment | Handling loose change/cards | Seamless online or cash-integrated options. |

## 🚀 Key Features

- **Smart Scheduling:** Never guess if the printer is occupied. Our real-time queue tracker lets you choose your perfect print window.
- **Universal Upload:** Supports PDF, DOCX, and common image formats. Your file, your way.
- **Campus-First Architecture:** Tailored for student life—fast, mobile-friendly, and lightweight.
- **Secure Transactions:** Encrypted payment gateways to keep your data safe while you focus on your studies.

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Icons:** Lucide React
- **State Management:** React Hooks
- **Animations:** CSS Transitions & Framer Motion

## 📦 Getting Started

### Prerequisites

Make sure you have Node.js installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/printfy.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd printfy
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Run the development server:**
   ```bash
   npm run dev
   ```

## 🎨 Design Philosophy

PrintFy was built with student accessibility in mind. We chose a vibrant, high-contrast color palette to ensure it feels energetic rather than administrative. The interface mimics the speed of the service: clean, direct, and distraction-free.
